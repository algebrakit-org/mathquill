MathQuill Foreign Object Test

Foreign Objects allow you to embed arbitrary HTML elements (buttons, inputs, custom components) directly into math expressions. The LaTeX only stores an ID reference (\foreignobject{id}), while the actual HTML element is managed separately.

Demo 1: Interactive Button

A clickable button embedded in a math expression.

Demo 2: Color Picker Widget

An HTML5 color picker embedded in the expression.

Demo 3: Stateful Counter Widget

A stateful counter widget that maintains its value independently of LaTeX.

Demo 4: Multiple Foreign Objects

Multiple different widgets in the same expression.
